# Christian TV IPTV

A curated collection of free Christian TV channels for IPTV players such as:

- TiviMate
- IPTV Smarters
- Televizo
- VLC
- Kodi
- OTT Navigator

## Features

- Free channels only
- Official streams whenever possible
- Russian
- Ukrainian
- English (planned)
- Automatically generated playlists
- Duplicate removal
- Weekly validation

## Playlists

| Playlist | Description |
|-----------|-------------|
| christian-all.m3u | All channels |
| christian-ru.m3u | Russian channels |
| christian-ua.m3u | Ukrainian channels |

## Project Goals

- Official streams only
- No piracy
- No paid IPTV
- Stable URLs
- Community maintained
